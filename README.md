# prog5-j2-lint
Ce projet utilise Python 3 et doit être analysé avec Pylint afin de vérifier la qualité du code et le respect des conventions de nommage.

Ce document explique comment :

installer l’environnement Python

installer Pylint

lancer l’analyse

## 1. Version de Python requise

Python 3.8 ou supérieur

## 2. Création et activation de l’environnement virtuel

python3 -m venv venv
source venv/bin/activate

## 3. Installation des dépendances

pip install pylint

## 4. Lancer Pylint sur un fichier

pylint nomFichier.py
